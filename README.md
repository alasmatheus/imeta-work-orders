# Inmeta Work Orders

Aplicativo React Native desenvolvido como teste técnico para a vaga de Desenvolvedor React Native Pleno na Inmeta.

## Objetivo

O projeto consiste em um aplicativo de gerenciamento de ordens de serviço com abordagem offline-first, utilizando armazenamento local com Realm e sincronização com API REST quando a conexão estiver disponível.

## Tecnologias utilizadas

- React Native
- TypeScript
- Realm.js
- Zustand
- Axios

## Funcionalidades

- Listagem de ordens de serviço armazenadas localmente
- Visualização de detalhes de uma ordem de serviço
- Criação e edição de ordens de serviço offline
- Sincronização de alterações locais com a API
- Atualização dos dados locais com informações mais recentes do servidor
- Tratamento de remoção lógica
- Estrutura preparada para resolução de conflitos de sincronização
